# Install tutorial videos

Drop the "Add to Home Screen" tutorial clips here. They'll be servable at
`/tutorials/<filename>` once the in-app install prompt code is wired up.

Suggested filenames (so the platform-detection code can find them):

- `install-ios.mp4`   — Safari → Share → Add to Home Screen
- `install-android.mp4` — Chrome → ⋮ menu → Install app

Optional poster images (first frame, shown before the video loads):

- `install-ios-poster.jpg`
- `install-android-poster.jpg`

Notes:
- Use `.mp4` (H.264) for broad support; `.webm` optional as a second source.
- Keep clips short (~15–20s) and encode muted-friendly (captions burned in).
